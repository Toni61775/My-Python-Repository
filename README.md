# My-Python-Repository


# 🐍 Python Fundamentals Project – Data Technician Bootcamp

## 📌 Overview
This project was completed as part of a Data Technician Bootcamp and demonstrates foundational Python programming skills. The exercises focus on building logical thinking, performing calculations, and creating interactive programs using core Python concepts.

---

## 🚀 Skills & Concepts Demonstrated

### ✅ Variables
- Used variables to store and manage data such as user input, calculations, and program state.
- Demonstrated understanding of data types including strings, integers, and floats.

### ✅ Input and Output
- Applied `input()` to collect user data dynamically.
- Used `print()` to display clear, formatted outputs and feedback to users.

### ✅ Type Casting
- Converted data types using functions like `int()` and `float()` to enable numeric calculations and comparisons.
- Ensured compatibility between user input and program logic.

### ✅ Conditional Logic (`if` Statements)
- Implemented decision-making using `if`, `elif`, and `else`.
- Built logic to handle different scenarios based on user input (e.g., correct vs incorrect access codes).

### ✅ Loops (`for` and `while`)
- Used `while` loops for repeated user interaction until conditions are met.
- Applied `for` loops for controlled iteration over sequences and ranges.

# 🛍️ Retail & Sales Data Analysis Project

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on using Python for real-world data analysis. The primary objective was to explore, clean, and analyse retail and sales datasets to uncover trends, patterns, and actionable insights.

Through this project, I developed practical experience in **data wrangling, filtering, and visualisation**, using industry-standard tools such as **pandas** and **matplotlib**.

---

## 🧰 Tools & Technologies
- Python
- pandas
- matplotlib
- Jupyter Notebook / Google Colab

---

## 🔍 Key Skills Demonstrated

### 📊 Data Exploration & Inspection
- Loaded and inspected datasets using `pandas`
- Explored dataset structure, data types, and summary statistics
- Identified missing values and inconsistencies

### 🧹 Data Cleaning & Preparation
- Handled missing data using:
  - `dropna()` to remove incomplete records
  - `fillna()` to replace missing values with meaningful defaults
- Ensured data quality and consistency for accurate analysis

### ✂️ Data Filtering & Subsetting
- Filtered datasets using conditional logic
- Selected specific rows and columns using:
  - `.loc[]` for label-based indexing
  - `.iloc[]` for position-based indexing
- Created subsets for targeted analysis (e.g. specific regions, products, or time periods)

---

### 🔄 Data Manipulation & Transformation
- Aggregated data using `groupby()` to identify trends across categories
- Organized data with `sort_values()` for better readability and insights
- Transformed datasets to support deeper analysis and reporting




## 📊 Project Focus

- Identifying high-performing and low-performing segments
